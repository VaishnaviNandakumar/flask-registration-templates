### Sample project to get started with Flask

### Set Up
Create Virtual Environment
```
py -3 -m venv venv
```
Activate
```
.venv\Scripts\activate
```
Install Flask
```
pip install Flask
```
To run
```
set FLASK_APP=sample.py
flask run
```
pip install flask-mysqldb
pip install Flask-WTF
pip install Flask-passlib
